월경 주기법
- 월경이 되풀이 되는 일정한 기간을 계산하여 피임하는 방법
- 월경이 시작된 날 ~ 다음 월경 시작 전날(보통 28 ~ 35일 사이)
- 여성 / 자연적 / 일시적

기초체온법
- 매일 기초체온을 측정하여 체온의 변화로 배란기를 예측하는 방법
- 기초 체온 : 충분한 수면을 취한 후 아침에 기상 하자마자 움직임 없이 측정한 체온
- 여성 / 자연적 / 일시적

질외 사정법
- 성관계 중 사정 전에 여성의 질에서 남성이 음경을 빼내 밖에서 사정하는 방법
- 사정 조절이 쉽지 않고 쿠퍼액(요도를 청소하기 위해 발기 때 나오는 액첼)에 정자가 섞여있을 수 있기 때문에 피임률이 낮음
- 남성 / 자연적 / 일시적

콘돔
- 전 세계에서 가장 널리 쓰이는 방법
- 장점 : 가격이 저렴하고 구하기 쉽다(편의점 구매 가능)
사용이 간편하다
인체에 무해하다
피임률이 높다
성병이 예방된다
- 물리적 / 일시적 / 남성
- 콘돔 사용시 유의점 : 유통기한을 지켜야 한다(최근 것이 좋음)
말린방향을 유의해야 한다(바깥쪽으로 말려야 함)
포장을 뜯을 때 콘돔이 찢어지지 않게 주의해야 한다
사정을 하자마자 버려야 한다(샐 수 있기 때문)
재사용하지 말아야 한다
끝을 비튼 채로 착용해야 한다(압력 때문에 터질 수 있기 때문)

페미돔
- 여성용 콘돔 같은 것
- 막힌 쪽은 링이 두 개, 뚫린 쪽은 링이 한 개
- 물리적 / 일시적 / 여성

경구(먹는) 피임약
- 여성호르몬을 통해 배란을 억제하는 방법
- 피임을 시작하려는 달의 월경 시작일부터 1일 1알 일정한 시간에 복용
- 21일 복용 -> 7일 휴약 -> 21일 복용을 반복
- 여성 / 화학적 / 일시적
피임 패치
- 경구피임약과 원리가 동일한 방법
- 경구피임약의 번거로움을 해소한 것
- 1장에 7일 동안 효과가 있음
- 단점 : 가격이 비쌈
부작용 가능성이 있음
- 여성 / 화학적 / 일시적

질정제(살정제)
- 성관계 전 질 안에 넣어 정자의 움직임을 저하시키고 사멸시키는 방법
- 사용하는 타이밍이 중요함(관계 1시간 전에 사용해야 함)
- 여성 / 화학적 / 일시적

자궁 내 장치(루프)
- 자궁 내에 삽입한 장치가 자궁 내막에 변화를 일으켜 수정란의 착상을 막는 방법
- 3 ~ 5년간 유지됨
- 구리 루프 : 구리 이온을 통해 자궁 내막에 염증을 일으켜 착상을 막음(생리혈이 많아지는 부작용이 있을 수 있음)
호르몬 루프(미레나) : 호르몬을 통해 자궁 내막을 두껍게 못 만들게 하여 착상을 막음(생리혈이 줄거나 월경이 중지되는 부작용이 있을 수 있음)
- 여성 / 화학적 / 반영구적

피하 이식술(임플라논)
- 호르몬이 분비되는 막대를 피부 아래에 이식하여 착상을 방해하는 방법(생리혈이 줄거나 월경이 중지되는 부작용이 있을 수 있음)
- 미레나와 같은 호르몬임
- 3년간 효과가 있음
- 여성 / 화학적 / 반영구적 

정관/난관 수술
- 정관 수술(남성) : 정소에서 정낭으로 연결되는 정관을 차단하는 방법
수술이 금방 끝남
정자가 없는 정액만 사정함
- 난관 수술(여성 : 나팔관으로 통하는 길인 난관을 차단하는 방법
복부를 절개해야 함
마지막 자녀를 재왕절개로 한 여성이 주로 함
배란/월경을 함
- 여성,남성 / 외과적 / 영구적

응급 피임약(사후 피임약)
- 성관계 후 응급으로 복용하여 인위적으로 임신을 막는 방법(배란, 수정, 착상을 모두 막음)
- 성관계 후 72시간 내에 복용해야 함
- 호르몬 농도가 매우 높기 때문에 부작용이 심함
- 일반적인 피임약이 아니고 몸에 매우 안좋기 때문에 최후의 방법으로 써야 함
